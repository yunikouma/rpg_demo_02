Quest Creator Download

The Quest Creator is a package that contains all the tools needed to create and test quests for Raining Chain.


How to Install Show

  • Download. Note: Google Chrome falsely considers the package as malicious. Try downloading on IE or Firefox. I'm currently working on fixing that.
  • Unzip.
  • Move the rc folder at C:\rc
  • Install nodejs_server.msi at C:\Program Files (x86)\nodejs
  • Install map_editor.exe at C:\Program Files (x86)\Tiled
  • Open the Quest Creator via the shortcut at C:\rc\QuestCreator.exe or via C:\rc\QuestCreator\QuestCreator.exe.
  • Check tutorials.
