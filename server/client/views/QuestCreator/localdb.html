To run Raining Chain on your computer, you can either use the public online database or a local database.
This is what you need to do in order to install and use a local database.

1. Install Mongodb
http://www.mongodb.org/

2. Create database named test on port 27017 (default port)
Should be connectable via localhost:27017/test

3. Create those collections:

db.createCollection("pingData");
db.createCollection("report");
db.createCollection("player");
db.createCollection("offlineAction");
db.createCollection("main");
db.createCollection("equip");
db.createCollection("account");
db.createCollection("competition");
db.createCollection("questRating");
db.createCollection("clan");
db.createCollection("highscore");
db.createCollection("questVar");
db.createCollection("mainQuest");
db.createCollection("contributionHistory");
db.createCollection("socialMedia");

4. Select Local Db below Start Server in the Quest Creator
